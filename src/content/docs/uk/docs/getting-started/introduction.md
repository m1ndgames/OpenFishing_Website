**OpenFishing** — це самостійно розміщуваний вебзастосунок для рибалок, щоб упорядковувати **колекцію приманок**,
позначати **рибні місця** на карті, записувати **улови** та відстежувати **спорядження** — усе з
єдиного застосунку з пріоритетом приватності, що працює на вашому власному сервері.

Це застосунок [SvelteKit](https://kit.svelte.dev/) на основі єдиної бази даних **SQLite**,
що постачається як один образ **Docker**. Немає жодного хмарного сервісу, облікового запису в третіх сторін і
жодної телеметрії: ваші фотографії та дані зберігаються на вашій інфраструктурі й ніде більше.

## Філософія

- **Самостійно розміщуваний і з пріоритетом приватності** — ви його запускаєте, дані належать вам. Нічого не залишає ваш сервер.
- **Без хмари** — для роботи ядра застосунку не потрібні зовнішні залежності.
- **Відкритий код** — випущено за ліцензією AGPL-3.0 на
  [GitHub](https://github.com/m1ndgames/OpenFishing).
- **Просто запускати** — один контейнер, один файл SQLite, два томи для бази даних і завантажень.

## Огляд можливостей

| Область | Що робить |
|---|---|
| [Приманки](/uk/docs/features/lures) | Фотобібліотека з тегами, видами, обраним, відстеженням загублених приманок, QR-мітками та швидким пошуком. |
| [Місця](/uk/docs/features/spots) | Інтерактивна карта Leaflet ваших рибних місць з GPS і фотографіями. |
| [Улови](/uk/docs/features/catches) | Записуйте вид, вагу, довжину, GPS, спіймав-відпустив і презентацію, з Індексом клювання на основі погоди. |
| [Індекс клювання](/uk/docs/features/bite-index) | Автоматична оцінка від 0 до 10, що визначає умови риболовлі за погодними даними. |
| [Спорядження](/uk/docs/features/tackle) | Вудлища, котушки, волосіні та комплекти, з історією шпуль. |
| [Статистика](/uk/docs/features/stats) | Тенденції у ваших уловах і спорядженні. |
| [Функції ШІ](/uk/docs/ai/ai-features) | Опційний чат-бот та розпізнавання риби й приманок за фотографіями. |
| [Багатокористувацький режим](/uk/docs/multi-user/authentication) | Опційний вхід для розміщення друзів і родини, з квотами та панеллю адміністратора. |
| [Посилання для поширення](/uk/docs/features/share-links) | Публічні сторінки лише для читання для приманки, місця чи улову. |
| [REST API](/uk/docs/api/rest-api) | JSON API лише для читання з токенами Bearer та інтерфейсом Swagger. |
| [Резервне копіювання та відновлення](/uk/docs/backup/backup-restore) | Експорт/імпорт ZIP одним кліком. |
| [Мови](/uk/docs/features/languages) | 9 мов із перемикачем прапорів. |

## Наступні кроки

- Дотримуйтесь [Швидкого старту](/uk/docs/getting-started/quick-start), щоб запустити однією командою.
- Розгорніть належним чином за допомогою [Встановлення (docker-compose)](/uk/docs/getting-started/installation).
- Налаштуйте поведінку за допомогою [Змінних середовища](/uk/docs/configuration/environment-variables).
