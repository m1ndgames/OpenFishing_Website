**Демо-режим** робить інстанцію OpenFishing **лише для читання**. Установіть змінну середовища **`DEMO_MODE`**
на будь-яке значення, щоб його ввімкнути (див.
[Змінні середовища](/uk/docs/configuration/environment-variables)).

## Що він робить

- **Усі записи заблоковано** — відвідувачі можуть переглядати все, але не можуть створювати, редагувати чи
  видаляти.
- **Банер** вказує, що інстанція є демо, а спроба запису показує **сповіщення toast**, яке пояснює, що записи
  вимкнено.
- **Зміна мови все ще працює**, тож відвідувачі можуть переглядати демо будь-якою з
  [9 мов](/uk/docs/features/languages).

## Публічне демо

Офіційне демо за адресою **[demo.openfishing.org](https://demo.openfishing.org)** працює в цьому режимі. Це
повністю заповнена інстанція, яку ви можете вільно досліджувати, не впливаючи на жодні дані.

```yaml
environment:
  - DEMO_MODE=1
```
