OpenFishing повністю інтернаціоналізований і постачається **9 мовами**, з **перемикачем прапорів** в інтерфейсі
для зміни мови будь-коли.

## Підтримувані мови

| Код | Мова |
|---|---|
| `EN` | Англійська |
| `DE` | Німецька |
| `FR` | Французька |
| `ES` | Іспанська |
| `IT` | Італійська |
| `NL` | Нідерландська |
| `PL` | Польська |
| `PT` | Португальська |
| `UK` | Українська |

Зміна мови працює навіть у [демо-режимі](/uk/docs/demo/demo-mode), де інші операції запису заблоковані.
