**Посилання для поширення** дають змогу опублікувати окрему **приманку**, **місце** чи **улов** як публічну
сторінку лише для читання — зручно, щоб похвалитися уловом або поділитися приманкою з другом, не надаючи йому
доступу до всієї вашої інстанції.

## Як це працює

- Створіть посилання для поширення для будь-якої окремої приманки, місця чи улову.
- Посилання відкриває сторінку **лише для читання**, що показує лише цей елемент.
- **Відкличте** посилання будь-коли, щоб знову зняти сторінку.

Розкривається лише конкретний елемент, яким ви ділитеся; решта ваших даних залишається приватною.

## Пов'язане

- Посилання для поширення доступні лише для читання, за духом подібні до [демо-режиму](/uk/docs/demo/demo-mode).
