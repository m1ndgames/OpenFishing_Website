**Бібліотека приманок** — це серце OpenFishing — візуальний каталог кожної приманки, яку ви маєте.

## Що можна відстежувати

- **Фотографії** — одне або кілька зображень на приманку, що зберігаються на вашому томі завантажень.
- **Теги та види** — категоризуйте приманки й зазначайте цільові види.
- **Обране** — позначайте приманки, до яких звертаєтеся найчастіше.
- **Відстеження загублених приманок** — позначте приманку як загублену, щоб вона зникла з вашої активної
  колекції без видалення її історії.
- **Кількість** — скільки одиниць певної приманки у вас є.
- **Послідовні номери** — кожна приманка отримує автоматичний послідовний ID (напр. `#0001`), зручний
  для [QR-міток](/uk/docs/features/qr-labels).

## Пошук приманок

- **Швидка фільтрація й пошук на боці клієнта** — фільтруйте за тегом, видом чи обраним і шукайте за назвою
  миттєво, без звернення до сервера.
- **Автопідказки** — поля тегів і видів пропонують уже використані значення.
- **Пагінація** — великі колекції гортаються плавно.

## Пов'язане

- Друкуйте [QR-мітки](/uk/docs/features/qr-labels) для непозначених приманок.
- Опублікуйте приманку публічно за допомогою [посилання для поширення](/uk/docs/features/share-links).
- Прив'яжіть приманку до [улову](/uk/docs/features/catches).
