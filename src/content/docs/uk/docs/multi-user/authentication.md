За замовчуванням OpenFishing працює в режимі **єдиного орендаря й повністю відкритому** — без входу. Установіть
змінну середовища **`ADMIN_PASSWORD`**, щоб увімкнути **багатокористувацький режим**.

## Увімкнення входу

Установіть `ADMIN_PASSWORD` (див. [Змінні середовища](/uk/docs/configuration/environment-variables)):

```yaml
environment:
  - ADMIN_PASSWORD=change_me
```

Після встановлення з'являється екран входу. Користувачі входять зі своєю **електронною поштою/іменем користувача
+ паролем**.

:::note
`AUTH_PASSWORD` — це **застаріла** резервна назва для `ADMIN_PASSWORD`. Використовуйте `ADMIN_PASSWORD`.
:::

## Обліковий запис адміністратора

- Обліковий запис **`admin`** завжди існує, коли автентифікацію ввімкнено.
- Його ім'я користувача завжди `admin`; його пароль — той, який встановлено в `ADMIN_PASSWORD`.
- Він **повторно синхронізується зі змінної середовища під час кожного запуску**, тож зміна змінної та
  перезапуск оновлюють пароль.

## Панель адміністратора

Адміністратори отримують панель за адресою **`/settings/admin`** для керування інстанцією:

- **Користувачі** — створення, увімкнення/вимкнення та керування обліковими записами.
- **Квоти сховища** — обмеження того, скільки кожен користувач може завантажити.
- **Перемикач чат-бота** — увімкнення/вимкнення [функцій ШІ](/uk/docs/ai/ai-features) для кожного користувача.
- **API-токени** — керування токенами Bearer [REST API](/uk/docs/api/rest-api).

## Ізоляція даних для кожного користувача

Приманки, місця, улови та спорядження кожного користувача **ізольовані** — користувачі бачать лише власні дані.

## Скидання пароля (SMTP)

Якщо ви налаштуєте SMTP (принаймні `SMTP_HOST` і `SMTP_FROM`), користувачі зможуть скидати забуті паролі
електронною поштою. Без налаштованого SMTP опція скидання прихована. Див.
[змінні SMTP](/uk/docs/configuration/environment-variables).

## Квоти сховища для кожного користувача

Адміністратори можуть встановлювати **квоти сховища** для кожного користувача з панелі адміністратора, щоб
жоден окремий користувач не міг заповнити диск завантаженнями.
