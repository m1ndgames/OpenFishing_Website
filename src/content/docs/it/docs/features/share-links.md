I **link di condivisione** ti permettono di pubblicare una singola **esca**, **spot** o **cattura** come pagina
pubblica di sola lettura — comodo per mostrare una cattura o condividere un'esca con un amico senza dargli
accesso all'intera istanza.

## Come funziona

- Crea un link di condivisione per qualsiasi singola esca, spot o cattura.
- Il link apre una pagina di **sola lettura** che mostra solo quell'elemento.
- **Revoca** il link in qualsiasi momento per rimuovere di nuovo la pagina.

Viene esposto solo l'elemento specifico che condividi; il resto dei tuoi dati resta privato.

## Correlati

- I link di condivisione sono di sola lettura, simili nello spirito alla [modalità demo](/it/docs/demo/demo-mode).
