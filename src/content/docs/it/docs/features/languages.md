OpenFishing è completamente internazionalizzato e distribuito in **9 lingue**, con un **selettore a bandiere**
nell'interfaccia per cambiare lingua in qualsiasi momento.

## Lingue supportate

| Codice | Lingua |
|---|---|
| `EN` | Inglese |
| `DE` | Tedesco |
| `FR` | Francese |
| `ES` | Spagnolo |
| `IT` | Italiano |
| `NL` | Olandese |
| `PL` | Polacco |
| `PT` | Portoghese |
| `UK` | Ucraino |

Il cambio di lingua funziona anche in [modalità demo](/it/docs/demo/demo-mode), dove le altre scritture sono
bloccate.
