As **ligações de partilha** permitem-te publicar uma única **isca**, **ponto** ou **captura** como uma página
pública só de leitura — útil para mostrar uma captura ou partilhar uma isca com um amigo sem lhe dar acesso a
toda a tua instância.

## Como funciona

- Cria uma ligação de partilha para qualquer isca, ponto ou captura individual.
- A ligação abre uma página **só de leitura** que mostra apenas esse item.
- **Revoga** a ligação a qualquer momento para retirar a página novamente.

Apenas o item específico que partilhas é exposto; o resto dos teus dados permanece privado.

## Relacionado

- As ligações de partilha são só de leitura, semelhantes em espírito ao [modo demo](/pt/docs/demo/demo-mode).
