O OpenFishing está totalmente internacionalizado e é distribuído em **9 idiomas**, com um **seletor de
bandeiras** na interface para mudar de idioma a qualquer momento.

## Idiomas suportados

| Código | Idioma |
|---|---|
| `EN` | Inglês |
| `DE` | Alemão |
| `FR` | Francês |
| `ES` | Espanhol |
| `IT` | Italiano |
| `NL` | Neerlandês |
| `PL` | Polaco |
| `PT` | Português |
| `UK` | Ucraniano |

A mudança de idioma funciona mesmo no [modo demo](/pt/docs/demo/demo-mode), onde outras escritas estão
bloqueadas.
