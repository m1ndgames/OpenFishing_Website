O OpenFishing pode gerar **etiquetas de código QR** para que uma isca física ligue diretamente à sua entrada na
tua biblioteca.

## Imprimir etiquetas

- As etiquetas têm o tamanho para material de etiquetas de **12,5 mm** — pequenas o suficiente para caber numa
  isca ou na sua caixa.
- Podes imprimir etiquetas para **iscas sem etiqueta** em lote.
- Cada etiqueta codifica uma ligação construída a partir do teu **`BASE_URL`**, de modo que a leitura abre a
  isca certa. Certifica-te de que `BASE_URL` está definido com o teu URL público real — consulta
  [Variáveis de ambiente](/pt/docs/configuration/environment-variables).

## Relacionado

- As etiquetas referenciam os números sequenciais `#0001` descritos em [Iscas](/pt/docs/features/lures).
