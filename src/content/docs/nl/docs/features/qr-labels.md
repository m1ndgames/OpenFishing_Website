title: QR-labels
description: Print kleine QR-code-labels voor je kunstaas om een fysiek kunstaas te koppelen aan de bijbehorende invoer.
---

OpenFishing kan **QR-code-labels** genereren zodat een fysiek kunstaas rechtstreeks verwijst naar de invoer in
je bibliotheek.

## Labels printen

- Labels zijn op maat voor **12,5 mm**-labelmateriaal — klein genoeg om op een kunstaas of het doosje te passen.
- Je kunt labels voor **ongelabeld kunstaas** in een batch printen.
- Elk label codeert een link die is opgebouwd uit je **`BASE_URL`**, zodat scannen het juiste kunstaas opent.
  Zorg dat `BASE_URL` op je echte publieke URL is ingesteld — zie
  [Omgevingsvariabelen](/nl/docs/configuration/environment-variables).

## Gerelateerd

- Labels verwijzen naar de volgnummers `#0001` die worden beschreven in [Kunstaas](/nl/docs/features/lures).
