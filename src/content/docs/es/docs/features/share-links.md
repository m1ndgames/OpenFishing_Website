Los **enlaces para compartir** te permiten publicar un solo **señuelo**, **punto** o **captura** como una página
pública de solo lectura — práctico para presumir una captura o compartir un señuelo con un amigo sin darle
acceso a toda tu instancia.

## Cómo funciona

- Crea un enlace para compartir de cualquier señuelo, punto o captura individual.
- El enlace abre una página de **solo lectura** que muestra únicamente ese elemento.
- **Revoca** el enlace en cualquier momento para retirar la página de nuevo.

Solo se expone el elemento específico que compartes; el resto de tus datos permanece privado.

## Relacionado

- Los enlaces para compartir son de solo lectura, similares en espíritu al [modo demo](/es/docs/demo/demo-mode).
