**Angelstellen** helfen dir, dich genau daran zu erinnern, wo du angelst. Jede Stelle liegt auf einer interaktiven
[Leaflet](https://leafletjs.com/)-Karte.

## Was du erfassen kannst

- **Kartenstandort** — setze eine Markierung auf die Karte oder gib GPS-Koordinaten direkt ein.
- **Fotos** — füge Bilder der Stelle hinzu.
- **Notizen** — beschreibe Zugang, Struktur oder Bedingungen.

## Nähe zu Fängen

OpenFishing verknüpft Stellen und Fänge nach Entfernung. Mithilfe der **Haversine**-Großkreisformel erscheint
jeder [Fang](/de/docs/features/catches), der **innerhalb von 100 m** einer Stelle erfasst wurde, automatisch
bei dieser Stelle — so entsteht ein Bild davon, welche Orte tatsächlich Fische bringen, ohne jeden Fang
manuell zuzuordnen.

## Verwandt

- Protokolliere einen [Fang](/de/docs/features/catches) mit eigenem GPS-Standort.
- Teile eine Stelle öffentlich mit einem [Freigabelink](/de/docs/features/share-links).
